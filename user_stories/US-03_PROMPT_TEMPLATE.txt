IMPLEMENTATION DETAILS

FUNCTION_SIGNATURE:
def group_back_to_back_messages(messages: List[ChatRequest]) -> List[List[ChatRequest]]:
    '''
    Group messages sent within time window that share context.

    Args:
        messages: List of ChatRequest objects in chronological order

    Returns:
        List of message groups (each group is a list of related messages)
    '''

INTEGRATION_POINT: /chat endpoint in main.py (before US-04)
EXECUTION_STEP: After US-02 role validation, before intent classification

DATA_SOURCES:
1. Message timestamps
2. Message content (for context detection)
3. Intent classification (for grouping logic)
4. User ID and chat session

ALGORITHM:
1. Sort messages by timestamp
2. Iterate through messages
3. Calculate time delta from previous message
4. If within window AND similar context → add to current group
5. If outside window OR different context → start new group
6. Return grouped messages
