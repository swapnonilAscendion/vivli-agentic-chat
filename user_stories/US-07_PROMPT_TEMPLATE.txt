3. HYBRID - Questions spanning multiple categories
4. ESCALATION - Issues requiring human intervention
5. UNKNOWN - Queries that don't fit any category

CLASSIFICATION RULES
================================================================================

RULE_1: Extract key terms and context from query
RULE_2: Match against intent taxonomy
RULE_3: Calculate confidence score (0.0-1.0)
RULE_4: Allow multi-intent classification when applicable
RULE_5: Set confidence thresholds for routing
