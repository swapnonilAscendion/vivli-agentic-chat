  Data Source: Vivli Data Request API
  Response Type: Real-time data with status

  Type: Multi-step routing
  Process: KB search first, then API if needed
  Response Type: Combined from both sources

  Type: Manual escalation
  Destination: Support team/Admin
  Response Type: Escalation message
