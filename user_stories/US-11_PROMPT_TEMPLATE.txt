EVALUATION METRICS
================================================================================

METRIC_1: Accuracy on Golden Dataset
  Definition: Percentage of correct answers
  Target: >= Baseline accuracy
  Measurement: Manual evaluation + automated scoring

METRIC_2: Hallucination Rate
  Definition: % of answers containing false information
  Target: < 5%
  Measurement: Expert review of sample responses

METRIC_3: Token Cost Per Response
  Definition: Average tokens used * cost per token
  Target: Competitive with or better than baseline
  Measurement: API usage logs analysis

METRIC_4: Latency (Response Time)
  Definition: Time from query to complete response
  Target: < 2 seconds average
  Measurement: API response time monitoring

METRIC_5: Groundedness Score
  Definition: % of responses grounded in provided context
  Target: >= 90%
  Measurement: Automated + manual verification

METRIC_6: User Preference
  Definition: User satisfaction with response quality
  Target: >= 4.0/5.0 rating
  Measurement: User feedback survey

================================================================================
EVALUATION FRAMEWORK
================================================================================

SCORING_FORMULA:
  Score = (Accuracy * 0.40) + (Cost_Score * 0.30) + (Latency_Score * 0.20) + (Hallucination_Score * 0.10)

WHERE:
  - Accuracy_Score: 0-1.0 (normalized)
  - Cost_Score: 0-1.0 (1.0 = equal/cheaper than baseline)
  - Latency_Score: 0-1.0 (1.0 = < 2s)
  - Hallucination_Score: 0-1.0 (1.0 = < 5% hallucination)

DECISION_RULE:
  IF (chosen_score > baseline_score AND hallucination_rate < 5%) THEN select_model
